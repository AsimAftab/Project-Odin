# Usage

Initialize Odin:

```powershell
odin init
```

Capture the workstation:

```powershell
odin snapshot
```

Open the dashboard:

```powershell
odin dashboard
```

Diagnose machine health:

```powershell
odin doctor
```

Preview restore:

```powershell
odin restore
```

Apply restore:

```powershell
odin restore --apply
```
